Functions Problem Set Exercises

  • function isEven
    1. takes a single number as an arguement
    2. checks if the number is even or odd
    3. returns true if the number is even and false if not
  • function factorial
    1. takes a single number
    2. times the number from 1 to the number
    3. 0 factorial is 1
  • function kebabToSnake
    1. takes a single string arguement that is a kebab case(dashs between words)
    2. transforms the kebab string into a snake case string
    3. Ex... convert-to-snake-case.....convert_to_snake_case

Please note NO console.log these are all RETURN statements
